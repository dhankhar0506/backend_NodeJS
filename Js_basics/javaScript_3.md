## What are Comments in JavaScript?

- Comments are lines in your code that JavaScript ignores.
- They are used to explain code, make it readable, or temporarily disable code.
  - Types of Comments in JS
    - Single-line Comment (//)
    - Multi-line Comment (/* ... */)

## Polyfills and transpilers?

- Polyfills and transpilers are essential tools for web developers to ensure their code works across different browsers and environments
- Polyfills: These are pieces of code that add missing features to older browsers.
- Transpilers: Transpilers convert code written in newer syntax (like ES6+) into older syntax (like ES5) that older browsers can understand. For instance, a transpiler would convert let name = 'Codetopology'; into var name = 'Codetopology'; for browsers that do not support the let syntax.
