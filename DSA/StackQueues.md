# Stack (LIFO) and Queue (FIFO)

## Stack - Last In First Out

A Stack is a linear data structure that follows the LIFO principle.

### Basic Operations
- **Push** — Add element to top
- **Pop** — Remove top element
- **Peek/Top** — View top element
- **isEmpty** — Check if stack is empty

### Key Concepts
- Operates on one end only (TOP)
- No random access
- Can be implemented using Arrays or Linked Lists

## Queue - First In First Out

A Queue is a linear data structure that follows the FIFO principle.

### Basic Operations
- **Enqueue** — Insert at rear
- **Dequeue** — Remove from front
- **Front/Peek** — View first element
- **isEmpty** — Check if queue is empty

### Types of Queues
- **Simple Queue** — Standard FIFO structure
- **Circular Queue** — Last position connects to first
- **Priority Queue** — Elements served based on priority
- **Deque (Double Ended Queue)** — Insert/delete from both ends
